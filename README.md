 Playwright Test for Events Widget

Описание проекта

Данный проект содержит автоматизированный тест для страницы:

https://dev.3snet.info/eventswidget/

Тест написан с использованием Playwright на языке JavaScript.

Цель теста — проверить базовую работоспособность интерфейса генератора виджета событий и корректность отображения превью после выбора параметров.

---

Что проверяет тест

Тест выполняет следующий сценарий:

1. Открывает страницу генератора виджета.
2. Выбирает тип события Affiliate.
3. Открывает список доступных источников.
4. Выбирает опцию "Выбрать все".
5. Нажимает кнопку "Сгенерировать превью".
6. Проверяет, что на странице появляется iframe с превью виджета.

Таким образом тест покрывает:

* загрузку страницы
* взаимодействие с элементами интерфейса
* генерацию превью виджета
* отображение iframe с результатом

Установка

1. Клонировать репозиторий

```
git clone <repository_url>
cd <repository_folder>
```

2. Установить зависимости

```
npm install
```

3. Установить браузеры Playwright

```
npx playwright install
```

---

Запуск тестов

Запустить тест:

```
npx playwright test
```

После выполнения тестов можно открыть HTML отчет:

```
npx playwright show-report
```

---

Наблюдения и особенности страницы

Во время разработки теста были обнаружены некоторые особенности поведения интерфейса:

* Некоторые элементы интерфейса используют кастомные dropdown-компоненты, которые требуют клика по специальному контейнеру `.checkselect-over`.
* При автоматической генерации теста через Playwright Codegen возникали ошибки `no context`, поэтому локаторы были уточнены вручную.
* Некоторые элементы интерфейса имеют нестабильную структуру DOM, поэтому используются более точные селекторы (`nth()` и текстовые локаторы).
